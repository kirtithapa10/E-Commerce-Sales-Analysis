# E-Commerce-Sales-Analysis
Target is a globally recognized retail brand based in the United States, known for offering customers exceptional value, inspiration, innovation, and a distinctive shopping experience. This dataset represents Target’s operations in Brazil and contains over 100,000 online orders placed between 2016 and 2018.

It provides comprehensive insights into various aspects of the business, including order status, pricing, payment methods, shipping performance, customer demographics, and product details. The goal of this analysis is to explore customer behavior, sales performance, and logistics efficiency to identify actionable insights that can help improve marketing strategies, delivery operations, and overall customer satisfaction.
# Feature
The data is available in 8 csv files
-customers.csv

-orders.csv

-sellers.csv

-products.csv

-order_items.csv

-payments.csv

-geolocation.csv
